# Git workflow

Her önemli değişiklikten sonra otomatik olarak `git commit` yap. "Önemli değişiklik" demek: yeni bir özellik, bug fix, refactor, bağımlılık güncellemesi, yapılandırma değişikliği veya birden fazla dosyayı etkileyen düzenleme. Küçük formatting/typo düzeltmeleri için bekleyip birleştirebilirsin.

- Commit mesajları Türkçe ve kısa olsun (özne + neden).
- Push işlemi için kullanıcı onayı bekle — sadece commit otomatik.
